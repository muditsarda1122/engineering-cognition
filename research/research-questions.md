# Open Research Questions

The following questions currently define the scope of Engineering Cognition research.

## Fundamental Questions

- What is Engineering Cognition?
- Does Engineering Cognition exist as a computational abstraction?
- How does it differ from memory?
- How does it differ from reasoning?

---

## Extraction

- Can engineering cognition be extracted automatically?
- Which observations should become cognition?
- Which observations should be discarded?

---

## Representation

- Does engineering cognition admit a canonical representation?
- Can multiple representations coexist?
- How hierarchical should cognition become?

---

## Evaluation

- Can engineering cognition improve long-horizon software engineering?
- How should accumulated understanding be measured?
- How should cognition quality be evaluated?

---

## Future Questions

- Can engineering cognition transfer between repositories?
- Can engineering cognition become collaborative?
- Can engineering cognition evolve autonomously?
