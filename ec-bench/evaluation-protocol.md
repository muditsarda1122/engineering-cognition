# Evaluation Protocol

Every EC-Bench experiment compares two environments.

Environment A

Stateless software engineering.

No accumulated engineering cognition.

Environment B

Engineering Cognition enabled.

The benchmark compares only one independent variable:

Availability of accumulated engineering cognition.

All other variables must remain identical.

---

## Repository

Both environments begin from the same repository state.

---

## Model

The same language model must be used in both environments.

---

## Prompt Order

Prompts must be presented in identical order.

---

## Evaluation

Every prompt output is evaluated independently.

Implementation prompts additionally execute repository verification.

---

## Scoring

Each answer receives independent scores across every benchmark metric.

Final benchmark scores are aggregated only after all prompts have completed.
